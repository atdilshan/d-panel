# D-Panel

A responsive dashboard built with Next.js and Material UI. It shows a grid of tiles. On desktop you get a sidebar and a modal for details, on mobile a bottom bar and a bottom sheet. Data is loaded from the appâ€™s API routes.

## Setup

1. Clone the repo
   git clone https://github.com/atdilshan/d-panel.git
   cd d-panel

2. Install dependencies
`pnpm install`
3. Run the app: `pnpm dev`
4. Open http://localhost:3000 in your browser.
